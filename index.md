# Indie Horror Games & Psychological Design

Indie horror games succeed by limiting player power, using atmosphere, and creating psychological pressure rather than relying on high-end graphics.

This documentation explores:
- Environmental storytelling
- Player vulnerability
- Minimalist horror mechanics
- Psychological tension in indie games
